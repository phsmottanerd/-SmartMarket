## 🎯 Sobre o Projeto

O **SmartMarket** é uma aplicação de console desenvolvida em **Java**, com foco na prática de conceitos fundamentais e intermediários de desenvolvimento backend.

O sistema simula o funcionamento de um mercado, permitindo o gerenciamento de produtos, compras e carrinho de compras através de uma interface de terminal.

Durante o desenvolvimento foram aplicados conceitos amplamente utilizados em sistemas corporativos, fortalecendo conhecimentos em **Programação Orientada a Objetos (POO)**, **Collections Framework**, organização de código e regras de negócio.
